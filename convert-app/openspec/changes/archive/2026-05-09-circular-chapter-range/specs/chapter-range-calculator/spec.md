## MODIFIED Requirements

### 需求：章数范围计算
系统必须计算两个选定圣经位置之间（含两端）的总章数。当起始位置在结束位置之后时，系统必须采用回绕模式：从起始位置读到全本末尾，再从开头读到结束位置。

#### 场景：同书卷、同章节
- **当** 起始位置为创世记1章，结束位置为创世记1章
- **则** 结果必须为1

#### 场景：同书卷、不同章节（非回绕）
- **当** 起始位置为创世记1章，结束位置为创世记3章
- **则** 结果必须为3

#### 场景：不同书卷（非回绕）
- **当** 起始位置为创世记49章，结束位置为出埃及记2章
- **则** 结果必须为 (50 - 49 + 1) + 2 = 4

#### 场景：回绕计算（跨书卷）
- **当** 起始位置在结束位置之后（例如启示录20章到创世记3章）
- **则** 系统必须计算从启示录20章到启示录末尾，再从创世记1章到创世记3章的总章数
- **则** 结果必须为 (22 - 20 + 1) + 3 = 6
- **则** 返回结果必须标记为回绕模式

#### 场景：回绕计算（同书卷）
- **当** 起始位置为创世记5章，结束位置为创世记3章（同书卷起止颠倒）
- **则** 系统必须采用回绕模式计算
- **则** 结果必须等于整本圣经减去创世记4章那一章

### 需求：结果显示
系统必须以人类可读的格式显示计算结果，同时展示两个位置和总章数。回绕模式下必须额外显示提示信息。

#### 场景：显示有效结果
- **当** 计算出有效的章数范围时
- **则** 显示格式必须为："<起始书卷><起始章>章 到 <结束书卷><结束章>章 = 共<总数>章"

#### 场景：显示回绕提示
- **当** 计算结果为回绕模式时
- **则** 除显示结果外，必须额外显示提示："已读完全本，从开头继续"
